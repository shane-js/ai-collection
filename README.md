# AI Collection

> **Disclaimer:** These projects are for demonstration and educational purposes only.

A collection of AI-powered CLI tools built with LiteLLM.

## Projects

**🎯 [ai-pitch-sales-meeting](./ai-pitch-sales-meeting)** — Generate sales pitches and meeting agendas by analyzing websites  
**🌐 [ai-web-browse](./ai-web-browse)** — Interactively browse and summarize websites

## Features

- ✅ Powered by LiteLLM for flexible LLM provider support
- ✅ Cost tracking with `--cost` flag (token usage and USD)
- ✅ Modern CLI tools built with Python 3.11+

## Quick Start

```bash
cd <project-name>
uv sync
uv run <command> --help
```

**Requirements:** Python 3.11+, [uv](https://docs.astral.sh/uv/), OpenAI API key (or other LLM provider supported by LiteLLM)

See individual project READMEs for detailed usage.
